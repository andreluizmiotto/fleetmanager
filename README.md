# fleetmanager
Fleet financial management application.

Due to the simplicity of the application, it was decided to avoid using frameworks.

Technologies:
  * Java (SDK 15.0.1);
  * JavaFX;
  * PostgreSQL;
  * Apache Maven;
  * JasperReports;
